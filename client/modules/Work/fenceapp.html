Crown Solar Fence Controller App

An Android App for Crown Solar Power Fence Controller Software. This is used by many security officials in major Government Organizations like ISRO,DRDL,BARC etc. With this application officer can monitor and control the Fence System anywhere anytime. Notifications appear right in the mobile if any breach or alerts occur in fence system.
